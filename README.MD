# 💈 Barba Nobre — Landing Page

Landing page moderna para uma barbearia premium, desenvolvida com foco em **design elegante**, **experiência do usuário** e **conversão de clientes**.

---

## 🚀 Sobre o Projeto

A **Barba Nobre** é uma landing page fictícia criada para simular o site de uma barbearia profissional. O projeto apresenta serviços, equipe, avaliações e um sistema simples de agendamento.

O objetivo é demonstrar habilidades em:

* HTML semântico
* CSS moderno (Flexbox + Grid)
* JavaScript para interatividade
* Estrutura de landing page voltada à conversão

---

## 🧩 Funcionalidades

* ✔️ Navegação com scroll suave
* ✔️ Seção de destaque (Hero)
* ✔️ Listagem de serviços
* ✔️ Galeria de imagens
* ✔️ Apresentação da equipe
* ✔️ Avaliações de clientes
* ✔️ Formulário de agendamento
* ✔️ Validação básica com feedback (toast)
* ✔️ Botão flutuante para WhatsApp

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Grid, Flexbox, variáveis CSS)
* JavaScript (DOM, eventos, validação)

## 📸 Galeria

A seção de galeria utiliza imagens reais para representar:

* Cortes masculinos
* Barba
* Ambiente da barbearia
* Produtos utilizados

> ⚠️ As imagens devem ser adicionadas manualmente na pasta `/img`.

---

## 📅 Agendamento

O formulário permite:

* Inserir nome e telefone
* Escolher serviço e barbeiro
* Selecionar data e horário

Ao confirmar:

* Exibe feedback visual (toast)
* Valida os campos obrigatórios
* Limpa o formulário automaticamente

---

## 📱 Responsividade

O layout é adaptado para:

* Desktop
* Tablet
* Mobile

Utilizando media queries para garantir boa experiência em diferentes telas.
